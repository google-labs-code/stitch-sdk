# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly.

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please report the vulnerability via
https://g.co/vulnz.

We will acknowledge receipt of your report within 3 business days and provide a
detailed response within 10 business days indicating next steps.
